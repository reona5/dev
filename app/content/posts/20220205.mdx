情報が混在しており、定期的に対応に追われ結構な時間を使うので備忘録。

## 結論

以下の issue にある設定を行う。
[How to use true colors in vim under tmux? #1246](https://github.com/tmux/tmux/issues/1246)

```vim
" 1. Edit .tmux.conf
set -g default-terminal
  - tmux-256color
set -ga terminal-overrides
  - ,*256col*:Tc

" 2. Edit: .vimrc
if exists('+termguicolors')
  let &t_8f =
  - \<Esc>[38;2;%lu;%lu;%lum
  let &t_8b =
  - \<Esc>[48;2;%lu;%lu;%lum
  set termguicolors
endif
```
